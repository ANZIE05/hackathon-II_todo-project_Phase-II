# Next.js Skill

## Description
Next.js is a React framework for building full-stack web applications. This skill provides expertise in Next.js development, including server-side rendering, static site generation, API routes, and modern React patterns.

## Capabilities
- Next.js project setup and configuration
- Pages and App Router patterns
- Server Components and Client Components
- Data fetching strategies (SSR, SSG, ISR, CSR)
- API routes and backend integration
- Authentication and middleware
- Styling solutions (CSS Modules, Tailwind, styled-components)
- Deployment and optimization
- TypeScript integration

## Usage Examples
- "Create a new Next.js project with TypeScript"
- "Set up API routes for user authentication"
- "Implement server-side rendering for SEO optimization"
- "Configure static site generation for blog posts"
- "Add middleware for request logging"
- "Optimize Next.js application performance"

## Best Practices
- Use the App Router for new projects
- Leverage Server Components for data fetching
- Implement proper error handling and loading states
- Follow Next.js caching strategies
- Use environment variables for configuration
- Implement proper TypeScript types for API routes
